CACHEIR HEALTH REPORT


INFORMATION AND HELP PAGE FOR CACHEIR HEALTH REPORT TOOL

HOW TO GENERATE AND USE A HEALTH REPORT:

1. Build SpiderMonkey's JavaScript shell.
2. Use environment variable: SPEW=RateMyCacheIR.
3. There are three ways to generate a CacheIR Health Report:

- Simply running a test case using --warp will produce a health report.
- In the shell you can call rateMyCacheIR() and a report will be generated for all scripts in the current zone.
- You may also call rateMyCacheIR() on a particular function to see the health report associated with that function's script. (ex. rateMyCacheIR(foo))

4. The health report will be named "spew_output.[PID].0", unless otherwise specified, and will be in the same directory you ran the shell in.
5. Go to CacheIR Health Report and input the file from the previous step into the tool.
